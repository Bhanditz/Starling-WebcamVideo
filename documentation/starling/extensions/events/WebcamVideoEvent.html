Classpublic class WebcamVideoEvent
InheritanceWebcamVideoEvent Event Object

A WebcamVideoEvent is triggered once per camera frame and is dispatched by WebcamVideo. They may come in handy if you want to handle the draw/upload process yourself; if you want to process a BitmapData/ByteArrays only if it gets updated (AR); or if you want to use a custom renderer for the video.
